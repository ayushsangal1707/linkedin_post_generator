# 🚀 LinkedIn Post Generator

An AI-powered tool that transforms your ideas into engaging, professional, and ready-to-post LinkedIn content in seconds.

---

## 📌 Overview

Creating high-quality LinkedIn posts consistently can be time-consuming. This project simplifies the process by using AI to generate well-structured, audience-focused posts from minimal input.

Whether you're a student, founder, or professional, this tool helps you maintain a strong LinkedIn presence effortlessly.

---

## ✨ Features

* 🧠 AI-generated LinkedIn posts
* ✍️ Converts short ideas into polished content
* 🎯 Multiple tones (professional, storytelling, motivational, etc.)
* ⚡ Quick and easy generation
* 📋 Copy-ready output
* 🔄 Regenerate posts instantly

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit** (for UI)
* **OpenAI API** (for content generation)

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/linkedin-post-generator.git
cd linkedin-post-generator
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Add your API key
   Create a `.env` file and add:

```
OPENAI_API_KEY=your_api_key_here
```

4. Run the app

```bash
streamlit run main.py
```

---

## 🧪 Usage

1. Enter your idea or topic
2. Select tone/style
3. Click **Generate**
4. Copy and post on LinkedIn 🚀
